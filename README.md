# DesignPattern
Head First Design Pattern 책 읽고 약간씩 변형해서 만들어본 패턴들
- 책에는 재밌는 패턴들이 정말 많이 더 있지만 `책읽기`와 `책정리`와 `구현`이 속도가 너무 느려서 먼저 책정리를 다 하고, 실제 구현은 마인크래프트 프레임워크를 만들 때 여러개를 적용해봤따.
